== Description ==

Obydullah Restaurant is a modern, elegant WordPress theme crafted specifically for restaurants, cafés, bistros, and food-related businesses. It combines a clean, professional layout with robust functionality, offering dedicated sections for menus, testimonials, chef specials, and contact details. The theme is fully responsive, ensuring your site looks stunning on all devices, and includes customizable options to tailor the look and feel to match your brand identity.

== Features ==

* Fully responsive and mobile-first design
* Customizer options for contact information (address, phone, email)
* Dedicated menu and testimonial sections
* Hero slider and event sections for promotions
* SEO-friendly semantic markup
* Lightweight and fast-loading architecture
* Custom navigation menu support

== Installation ==

1. Upload the `obydullah-restaurant` folder to the `/wp-content/themes/` directory.
2. Activate the theme through the 'Appearance > Themes' menu in WordPress.
3. Install and activate any recommended plugins.
4. Customize your site's appearance via 'Appearance > Customize'.

== Recommended Plugins ==

* Contact Form 7 – for adding contact forms
* Obydullah Restaurant Core – for additional custom post types and shortcodes

== Customization ==

You can easily customize the theme by navigating to:
Appearance → Customize → Contact Info

This section allows you to update your restaurant's address, phone number, and Opening Hours.
